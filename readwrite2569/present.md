# Prompt สำหรับสร้าง Presentation แนะนำระบบประเมินผลนักเรียน

**คำสั่งสำหรับ Claude:**
กรุณาสร้างโครงร่างสไลด์นำเสนอ (Presentation) เพื่อแนะนำและสอนการใช้งาน "ระบบประเมินผลนักเรียน (Admin Edition)" โดยอ้างอิงจากข้อมูลของระบบด้านล่างนี้ ขอให้แบ่งเป็นสไลด์แต่ละหน้า พร้อมระบุหัวข้อสไลด์ (Title) เนื้อหาที่จะโชว์บนสไลด์ (Bullet points) และบทพูดสำหรับผู้นำเสนอ (Speaker notes) ให้ชัดเจน เข้าใจง่าย โดยให้มีโทนการนำเสนอที่น่าสนใจ และเหมาะสำหรับนำไปพูดบรรยายให้คุณครูและแอดมินฟัง

---

### **ข้อมูลเกี่ยวกับระบบประเมินผลนักเรียน (สำหรับเป็นบริบทให้ Claude):**

**1. ภาพรวมระบบ**
*   **ชื่อระบบ:** ระบบประเมินผลนักเรียน (บันทึกคะแนน แปลผล และสรุปรายงานผลการเรียนอย่างเป็นระบบ)
*   **ผู้พัฒนา:** ธุรการไทยงาม
*   **กลุ่มเป้าหมาย:** คุณครูประจำชั้น และ ผู้ดูแลระบบ (Admin)
*   **เทคโนโลยี:** Web Application ทันสมัย รองรับทั้งคอมพิวเตอร์และมือถือ มีระบบ Dark Mode

**2. ฟีเจอร์หลักในระบบ**
*   **หน้าเข้าสู่ระบบ (Login):** ดีไซน์สวยงามทันสมัย แยกสิทธิ์การใช้งานระหว่างคุณครูและแอดมินอัตโนมัติ
*   **ระบบเมนู 2 ส่วนหลัก (Tabs):**
    1.  **เมนู "บันทึก" (Data Entry)**
        *   ตารางรายชื่อนักเรียนพร้อมช่องกรอกคะแนนตามหัวข้อประเมิน
        *   **ระบบอัจฉริยะ:** แปลงคะแนนเป็นระดับคุณภาพ (ดีมาก, ดี, พอใช้, ปรับปรุง) และรวมคะแนนให้อัตโนมัติทันทีที่พิมพ์
        *   **การค้นหา:** ค้นหานักเรียนด้วยชื่อหรือรหัสประจำตัวได้อย่างรวดเร็ว
        *   **เครื่องมืออำนวยความสะดวก:**
            *   ดูเกณฑ์การประเมินได้ทันที
            *   **ทำงานแบบออฟไลน์:** สามารถดาวน์โหลดไฟล์ฟอร์ม Excel ไปกรอกคะแนน แล้วนำมาอัพโหลดเข้าระบบได้
            *   บันทึกข้อมูลเก็บไว้บนคลาวด์ได้อย่างปลอดภัย
    2.  **เมนู "สรุป" (Statistics)**
        *   ตารางสรุปจำนวนนักเรียนที่ได้ระดับคุณภาพต่างๆ แยกตามหัวข้อ
        *   กราฟสถิติ (Charts) สวยงาม เข้าใจง่าย
        *   **การออกรายงาน (Export):** สามารถส่งออกรายงานเป็นไฟล์ PDF ประจำห้องเรียน และส่งออกข้อมูลเป็น Excel ได้

**3. ฟีเจอร์พิเศษสำหรับผู้ดูแลระบบ (Admin Features)**
*   **Admin Dashboard:** หน้ากระดานสรุปความคืบหน้าการกรอกคะแนนของ "ทุกห้องเรียน" ในโรงเรียน พร้อมแถบเปอร์เซ็นต์ความคืบหน้า
*   **ดูข้อมูลข้ามห้อง:** แอดมินสามารถเลือกดูและจัดการข้อมูลของห้องเรียนใดก็ได้ผ่าน Dropdown เมนู
*   **รายงานระดับโรงเรียน:** สามารถส่งออกรายงานสรุปสถิติแบบ PDF ในภาพรวมของทั้งโรงเรียนได้
*   **จัดการรายชื่อ:** ปุ่ม "อัปเดตรายชื่อ" สำหรับซิงค์รายชื่อนักเรียนล่าสุดจากฐานข้อมูลหลักเข้าสู่ระบบ โดยไม่กระทบคะแนนเดิมที่กรอกไว้

### **โครงสร้างสไลด์ที่ต้องการ (แนะนำ):**
*   **Slide 1:** Title (หน้าปก - แนะนำชื่อระบบ)
*   **Slide 2:** Overview (ระบบนี้คืออะไร? ช่วยแก้ปัญหาอะไรให้คุณครู?)
*   **Slide 3:** Getting Started (การเข้าสู่ระบบ)
*   **Slide 4:** Teacher Dashboard - Data Entry (การบันทึกคะแนนแบบปกติ)
*   **Slide 5:** Smart Features (ฟีเจอร์โหลด/อัพโหลด Excel และการแปลผลอัตโนมัติ)
*   **Slide 6:** Reports & Analytics (การดูสถิติแท็บสรุป และการดาวน์โหลด PDF/Excel)
*   **Slide 7:** Admin Zone (ฟีเจอร์สำหรับผู้ดูแลระบบ - Dashboard โรงเรียน, อัปเดตรายชื่อ)
*   **Slide 8:** Q&A / บทสรุป

*(หมายเหตุสำหรับ Claude: สามารถปรับแต่งโครงสร้างได้ตามความเหมาะสมเพื่อให้การนำเสนอลื่นไหลที่สุด)*
